# 🐴 Furto

### 📜 Regras Gerais:
- Furto de cavalos, carroças, barcos ou balões é uma infração grave.
- Após ser vítima de furto, só é permitido rastrear ou chamar o cavalo/carroça após 15 minutos.

### 🔪 Condições para Furto:
- Deve ser realizado de forma furtiva; o uso de força bruta é proibido.
- Ações de furto devem ser realizadas individualmente.
