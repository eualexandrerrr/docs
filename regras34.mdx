# 🏦 Regras para Roubo a Bancos

### 📜 Requisitos para o Assalto:
- É obrigatório que os bandidos estejam armados de maneira adequada.
- Todos os participantes do assalto devem portar armas.

### 📢 Procedimentos e Regras:
- Os oficiais têm permissão para intervir após o término do assalto e podem prender os criminosos com as mãos ao alto.
- É necessário aguardar a resposta da patrulha local antes de prosseguir, garantindo que os reféns sejam devidamente negociados.
- Durante a negociação, a quantidade de reféns deve ser claramente declarada.
- Todo tipo de negociação realizada é valida para ambos lados da ação.

### 📝 Após o Assalto:
- O bandido que derrotar os oficiais pode ser adicionado à lista de procurados caso haja testemunhas.

### 👥 Número de Participantes:
- **Mínimo:** 5.  
- **Máximo:** 10.

### 🔫 Regras Adicionais:
- **Armas:** Todos os participantes podem estar armados.  
- **Tempo de espera:** Aguardar 30 minutos para os oficiais chegarem, caso não cheguem pode finalizar a ação e ir embora com o dinheiro/itens.
