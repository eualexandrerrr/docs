# 🏪 Roubo a Lojas/Saloons/Igrejas/Cinemas/Teatro

### 📜 Procedimentos:
- Deve-se esperar 20 minutos para a chegada da polícia e início das negociações.
- Apenas um bandido pode permanecer do lado de fora como vigia.
- É obrigatório determinar o número de reféns no início da negociação.

### ❌ Restrições:
- Não é permitido matar ou ferir os negociadores.
- O bandido negociador que estiver usando máscara deve se identificar para garantir a segurança dos reféns.

### 🛑 Regras Finais:
- Caso o bandido saia do local, ele não poderá retornar para negociar novamente.
- Após o roubo, é permitido solicitar uma fuga para outra troca de reféns.
- Há um tempo máximo estabelecido para a fuga, após o qual os oficiais podem abrir fogo se necessário.
- Durante as negociações, tanto bandidos quanto oficiais devem garantir que a ação seja bem registrada.

### 👥 Número de Participantes:
- **Mínimo:** 4.  
- **Máximo:** 8.

### 💼 Regras Adicionais:
- **Armas:** Todos podem estar armados.  
- **Loot de Oficiais:** Não permitido.
- **Tempo de espera:** Aguardar 20 minutos para os oficiais chegarem, caso não cheguem pode finalizar a ação e ir embora com o dinheiro/itens.
