# ⚕️ Normas Específicas das Clínicas

### 👔 Vestimenta:
- É proibido alterar o uniforme oficial, exceto para adicionar óculos, chapéus e colares.

### 🚫 Conduta:
- Corrupção em qualquer forma é inaceitável para personagens médicos.
- Como médico, você não possui autoridade legal para deter, prender ou acusar indivíduos.
